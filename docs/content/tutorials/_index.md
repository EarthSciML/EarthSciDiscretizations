End-to-end guides for contributors. Each tutorial picks a concrete artifact in
the repo as a running example and walks through the authoring pipeline that
produced it, with runnable commands and pointers to the canonical files.

- **[Add a new discretization rule]({{< ref "/tutorials/add-a-rule" >}})** —
  declare the rule JSON, ship canonical and convergence fixtures, register
  with the walker harness, and render the catalog page. Worked example:
  `centered_2nd_uniform` on a Cartesian axis.
